Trello Application
Overview
This project is a Trello application built on .NET 8 using MediatR CQRS.
It allows users to manage tasks, create boards, and collaborate effectively.

Features
  * Create and manage boards
  * Add and organize tasks within boards
  * Track personal task progress

Technologies Used
  * .NET 8
  * MediatR CQRS
  * Entity Framework Core
  * ASP.NET Core WebApi
  * SQL Server
  * Angular (for frontend)
